# jethbot
Discord Bot Using All Of Jeth's Acoustic_Mafia Messages As  A Dataset


DESCRIPTION

A discord.py bot interfaced with all messages sent by Jeth in Acoustic_Mafia discord server.

1. Returning A Random Messages From JSON Dataset

2. Returning A Specified Multiple Amount Of Messages From JSON Dataset

3. Querey Dataset For Matches And Return 8 Matches (Amount Of Matches Can Be Changed Easily In Script)

4. Send A Random Message To The Chat Hourly (Amount Of Time Can Be Changed Easily In Script)




INSTALLATION:

1: git clone https://github.com/jujijoog/jethbot

2. cd jethbot

3. Replace Channel ID's with the channel you want bot to operate in (two instances)  

4. Replace "Token Goes Here" with your bot's token


*Note: This bot can also easily be adapted to work with any other datasets in json format. To do this just point to the correct json file and change the objects specified in the data definition.



JETH BOT COMMAND LIST ----

MAIN COMMANDS: 

!jeth -- Returns A Random Message From Dataset 

!jethx * -- Return A Specified Amount Of Messages From Dataset

!search *  -- Query And Return Up To 10 Results From The Messages Dataset
